## üöÄ Getting Started

Follow these steps to set up the project locally:

### ‚úÖ Prerequisites
- You‚Äôll need a **free API key** from [OpenWeatherMap](https://home.openweathermap.org/).
  1. Create a free account  
  2. Navigate to **API keys** tab  
  3. Copy your generated key  

---

### üì• Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Madhusudan043337/climacast-weather-app.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd climacast-weather-app
   ```

3. **Add your API Key**
   - Open `script.js`
   - Locate the `apiKey` constant (line 2)
   - Replace the placeholder with your actual API key  

4. **Run the application**
   - Open `index.html` directly in your browser  

   > ‚ö†Ô∏è For the **geolocation feature**, you may need to run via a local server (e.g., VS Code [Live Server extension]) since some browsers block geolocation on `file:///` URLs.
